#### Usage:
1 - Copy .env.example to .env

```
cp .env.example .env
```

2 - edit your .env with the required values:
```
NODE_ENV=development

CLIENT_ID=
CLIENT_SECRET=
CLIENT_TOKEN=

DEFAULT_PREFIX=!
```
3 - build the project

```
yarn run build
```

4 - run the bot

```
yarn run start
```
